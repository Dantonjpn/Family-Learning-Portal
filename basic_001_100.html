★今週のトレーニング：長い文を作る
接続詞（When, If, Because）を使うと、2つの文を1つにつなげることができます。「～のとき、…だ」「もし～なら、…だ」というセットのリズムを覚えましょう。

私は / 知っています / （ということ）を / 彼が / 有名な医者である [接続詞that] テーマ43 p.98
023 When I came home, my mother was cooking.
（～のとき）私が / 帰宅したとき / 私の母は / 料理をしていました [接続詞when] テーマ45 p.102
024 If it is sunny tomorrow, let's go fishing.
もし～なら / （天気）晴れ / 明日 / （しましょう）行こう / 釣りに [接続詞if] テーマ46 p.104
025 Because I was tired, I went to bed early.
（なぜなら）私は疲れていたので / 私は / 寝ました / 早く [接続詞because] テーマ46(参考)
026 I stayed home because it was raining.
私は / いました / 家に / （なぜなら）雨が降っていたので [接続詞because] テーマ46(参考)

教えてください / 私に / いつ / そのバスが / 到着するか [間接疑問(先行学習)] テーマ86 p.186
029 Wash your hands before you eat dinner.

★今週のトレーニング：比べる表現
「AとBを比べる（er / more）」、「Aが一番（the est / the most）」。「より～」と「最も～」の使い分けをマスターしましょう。長い単語（interesting, famousなど）はmore/mostを使います。
